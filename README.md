# Sticky Notes

Leave notes on any website and they will still be there the next time you visit. The notes are tied to the page URL so your reminder on a specific GitHub issue or a note on a documentation page stays exactly where you left it.

## What it does

You can add as many sticky notes as you want to any page. Each note is draggable so you can move it wherever it is not in the way. Notes come in different colors so you can organize them visually. Everything is saved automatically and persists between sessions without you having to do anything.

## How to use it

Click the extension icon while on any page to add a new note. A note will appear on the page that you can click to edit, drag to reposition, or close when you are done with it. Next time you visit that URL your notes will load back automatically.

## How to install

Clone or download this repo, open Chrome and go to chrome://extensions, enable Developer Mode, click Load unpacked, and select this folder.

## Permissions it uses

It needs storage to save your notes, scripting and activeTab to inject the notes onto the page you are viewing. None of your note content is sent anywhere.

## Built with

Manifest V3, vanilla JavaScript, HTML and CSS using Chrome's storage, scripting, and activeTab APIs.
